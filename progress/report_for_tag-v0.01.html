ai-embedded-lab Progress Report

Tag: v0.01

Goal

Make the full pipeline work end-to-end: code generation → compile → load/flash via GDB → run → verify, with clean output (no SSL or GDB warnings).

What Was Fixed/Added

  • Flash step enabled and corrected: switched BMDA GDB adapter from flash to load, reordered file before attach, and increased flash timeout to 120s.
  • SSL warning suppression added: new toggle so warnings are suppressed only when SSL verification is disabled.
  • ESP32JTAG config extended: added web settings and warning suppression option.

Files Changed

  • adapters/flash_bmda_gdbmi.py
  • adapters/observe_gpio_pin.py
  • adapters/preflight.py
  • configs/esp32jtag.yaml
  • tools/la_verify.py

Final Command

python orchestrator.py run \
  --probe configs/esp32jtag.yaml \
  --board configs/boards/rp2040_pico.yaml \
  --wiring "swd=P3 reset=P2.0 verify=P0.1"
