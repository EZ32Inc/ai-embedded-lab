# Generated Example First Execution Enablement Batch v0.1

## Purpose

This note defines the first execution-enablement batch after governance has
reached its current stop point.

## Recommended Batch

Do one of these as a small bounded batch:

1. Provision one UART runtime path and run one generated UART example.
2. Complete one ADC external-input contract for one generated ADC example.

## Preferred Order

If hardware setup can be changed now:
- provision one RP2 or STM32 UART path first

If hardware setup cannot be changed now:
- complete one ADC external-input contract first

## What This Batch Should Not Try To Do

- broad multi-example runtime execution
- broad ESP32 generated-example runtime promotion
- new family expansion
- another governance-only round
