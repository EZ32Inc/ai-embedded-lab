# Generated Example First Non-Governance Batch v0.1

## Purpose

This note defines the next batch after the current generated-example governance
work reaches its stop point.

## Recommended First Non-Governance Batch

Choose one of these two and do it as a small bounded execution-facing batch:

1. Provision one UART runtime path and run one generated UART example live.
2. Complete one ADC external-input contract for one generated ADC example.

## Preferred Order

If hardware provisioning is available:
- start with one RP2 or STM32 UART path

If provisioning is not available:
- complete one ADC external-input contract first

## Why

These two options remove real blockers.

They are better than:
- adding more governance-only notes
- broad runtime-validation campaigns
- broad new-family expansion

- broad ESP32 generated-example runtime claims
